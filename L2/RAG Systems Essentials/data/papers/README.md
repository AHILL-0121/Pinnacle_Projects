# Papers Directory

Place your AI research paper PDFs here.

## Recommended Papers

1. **Attention Is All You Need** (Transformer)
   - https://arxiv.org/abs/1706.03762
   
2. **Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks** (RAG)
   - https://arxiv.org/abs/2005.11401
   
3. **Language Models are Few-Shot Learners** (GPT-3)
   - https://arxiv.org/abs/2005.14165

## How to Get Papers

1. Go to arxiv.org
2. Search for the paper ID (e.g., 1706.03762)
3. Click "PDF" to download
4. Save to this directory

## File Naming

Name your files descriptively:
- `attention_is_all_you_need.pdf`
- `rag_paper_2020.pdf`
- `gpt3_language_models.pdf`

The system will extract the paper title from the filename if not detected in the PDF.
